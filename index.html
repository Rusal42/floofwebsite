Gambling & Games

Exciting gambling commands, daily rewards, and interactive games to keep your community engaged.

Gambling commands:
%balance View your current coins.
%beg Chance to receive a small coin reward.
%work Earn coins on a cooldown.
%slots [amount] Slot machine with wins/losses based on symbols.
%coinflip [amount] [heads|tails] 50/50 coin toss bet.
%blackjack [amount] Play blackjack vs the dealer. Hit/Stand via prompts.
%roulette [amount] [color|number] Bet on red/black/green or a number.
%donate @user [amount] Send coins to another user.
%leaderboard See top balances.
%levels Check XP/leveling progress.
%release Claim periodic release rewards.
%beatup [@user] Fun combat-themed coin interaction.
%gambling Opens the gambling menu.

Moderation Tools

Powerful moderation features including setup commands, role management, and server configuration.

Moderation commands:
%ban @user [reason] Ban a user from the server.
%kick @user [reason] Kick a user from the server.
%timeout @user [duration] [reason] Timeout a user for a period.
%warn @user [reason] Warn a user; logs to infractions.
%infractions @user View a user's infractions.
%clearinfractions @user Clear a user's infractions.
%slowmode [seconds] Set channel slowmode.
%purge [1-100] Bulk delete recent messages in the current channel.
%ticket Ticket system menu and actions.
%advancedlog Configure/view advanced logging.
%antispam Anti-spam controls and toggles.
%whois @user View detailed user info.
%av @user Show user's avatar.
