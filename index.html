Moderation Tools

Powerful moderation features including setup commands, role management, and server configuration.

Moderation commands (enforced by Discord permissions):
%ban @user [reason] Ban a user from the server. Requires: Ban Members
%kick @user [reason] Kick a user from the server. Requires: Kick Members
%timeout @user [duration] [reason] Timeout a user. Requires: Moderate Members
%warn @user [reason] Warn a user; logs to infractions. Requires: Manage Messages
%infractions @user View a user's infractions. Requires: Moderator
%clearinfractions @user Clear a user's infractions. Requires: Administrator
%slowmode [seconds] Set channel slowmode. Requires: Manage Channels
%purge [1-100] Bulk delete recent messages. Requires: Manage Messages
%ticket Ticket system menu and actions. Requires: Manage Server
%advancedlog Advanced logging configuration. Requires: Manage Server
%antispam Anti-spam controls and toggles. Requires: Manage Server
%createlevelroles [start] [end] Create level roles automatically. Requires: Administrator
%whois @user View detailed user info.
%av @user Show user's avatar.

Only members with the listed Discord permissions can execute these actions. The bot also needs these permissions in the server/roles.

Configuration

Simple setup process with helpful guides and commands to get your server running quickly.

Admin & Configuration (Administrator required):
%config modlog #channel Set the channel for moderation logs. Requires: Administrator
%config roles #channel Set the role selection/menu channel. Requires: Administrator
%config welcome #channel Set the welcome messages channel. Requires: Administrator
%config gambling #channel Restrict gambling to a specific channel. Requires: Administrator
%config changelog #channel Set the changelog announcements channel. Requires: Administrator
%config ruleschannel #channel Set default channel for Rules/Agree menu. Requires: Administrator
%config colorschannel #channel Set default channel for Color Roles menu. Requires: Administrator
%config rulescontent title:"Title" description:"Line1\nLine2" footer:"Footer" button:"Agree" role:"Member" Customize rules menu content and assign role name. Requires: Administrator
%config colorcontent title:"Pick a Color" description:"Choose a color role" Customize color menu embed text. Requires: Administrator
%config rulesmenu [#channel] Post the rules/agree menu (uses saved defaults if omitted). Requires: Administrator
%config colormenu [#channel] Auto-create color roles if missing and post the color roles menu (uses saved defaults if omitted). Requires: Administrator
%config prefix [symbol] Change the server default prefix. Requires: Administrator
%config view View all current server settings.
%config reset [ruleschannel|colorschannel|rulescontent|colorcontent] Reset specific legacy menu settings. Requires: Administrator
%config reset Reset all configuration to defaults. Requires: Administrator
%vcconfig Configure temporary voice channels. Requires: Manage Channels
%setup Guided setup for moderation, logging, and tickets. Requires: Manage Server
%levels config Configure the leveling system. Requires: Administrator
%levels reset @user Reset a user's XP/level. Requires: Administrator
%levels reset all Reset all users' XP/levels. Requires: Administrator
Directions:
  1. Set default channels: %config ruleschannel #rules, %config colorschannel #roles.
  2. Optionally customize text:
    %config rulescontent title:"Server Rules" description:"1. Be nice\n2. No spam" footer:"Click to agree" button:"I Agree" role:"Member"
    %config colorcontent title:"Pick a Color" description:"Choose a single color role"
  3. Post menus using saved defaults: %config rulesmenu, %config colormenu. You can override channel inline, e.g. %config rulesmenu #announcements.
  4. Review current settings: %config view. Reset fields if needed: %config reset rulescontent.

All admin/config commands validate permissions. Non-admins cannot change server settings. For legacy menus, ensure the bot has: Manage Roles (to assign the agree role and color roles), Send Messages, Embed Links, and Add Reactions/Use External Emojis (if used) in the target channels.
